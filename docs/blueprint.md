# **App Name**: ShopSync

## Core Features:

- Shop Registration: Allow shop owners to register and receive a unique Shop ID.
- Product Management: Enable shop owners to add, edit, and delete products with details like name, description, SKU, category, price, stock, and images.
- Order Management: Allow customers to browse products, add to cart, and checkout with options for online payment or cash on delivery.
- Order Status Updates: Enable shop staff to update the order status (Pending, Accepted, Preparing, Out for Delivery, Delivered) and notify customers.
- Delivery Charge Management: Set and manage delivery charges per shop and adjust them per order.
- Staff Activity Logging: Record staff activities, generate an LLM report and analytics with key metrics from logs, providing insights for owners. The LLM tool analyzes actions like order updates and product modifications.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust and reliability in transactions.
- Background color: Light gray (#F5F5F5), nearly white, to provide a clean and neutral backdrop that highlights the products.
- Accent color: Vivid orange (#FF9800) to draw attention to important actions and promotions.
- Body and headline font: 'PT Sans' for a balance of warmth and legibility across all content.
- Use clear and consistent icons to represent different product categories and actions within the app.
- A clean, grid-based layout to present product listings and shop information clearly.
- Subtle transitions and animations to enhance user engagement, like loading indicators and smooth scrolling.
